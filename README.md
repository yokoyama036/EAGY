# 開発言語
- Ruby 3.0.1
- Rails 6.1.7.3
# 就業Termの技術
- devise
- rails_admin
- cancancan
# カリキュラム外の技術
- Active Storage
- chat GPT API
# 実行手順
```
$git clone git@github.com:yokoyama036/EAGY.git
$cd EAGY
$bundle install
$yarn install
$rails db:create && rails db:migrate
$rails s
```
